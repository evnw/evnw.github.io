Naturalistic Driving App

Developed as a capstone project for the Western Australian Centre for Road Safety Research (WACRSR) to be used in naturalistic driving studies. It collects user GPS and high frequency accelerometer data, securely transmitting to Cloud Firestore with user consent. Additional features include user info collection, a statistics page, and a tutorial.

Hotter/Colder

Developed as an 'Agile Web Development' project to be a Wordle-like web game. Hotter/Colder mimics the childhood game by tasking the player to find a square in a 10x10 grid, informing them if they are 'hot' or 'cold' along the way.
