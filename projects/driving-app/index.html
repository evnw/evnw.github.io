Naturalistic Driving App

This app was developed as a capstone project using React Native Expo for the Western Australia Centre for Road Safety Research (WACRSR). It since has been used to collect data for naturalistic driving studies. It collects user GPS and high frequency accelerometer data, securely transmitting it to a Cloud Firestore database with user consent. Additional features include user info collection, a statistics page, and a tutorial.
